FAQ

What is Good Old Downloads (GOD)?

A site which automates the process of sharing games from GOG.com

How does it work?

Simply vote on games you want and the site will upload them in the order of which received the highest number of votes. Games are manually checked each day for updates then flagged for upload.

Why can't I vote on a game?

The game was recently voted on and re-uploaded.

How do I download or extract files?

We recommend JDownloader 2 to download and 7-Zip to extract files.

Which file hosters are free and/or fast?!

Google Drive is the best. Next best are Openload.co, zippyshare (only for files under 500MB), filecloud.io, and Filescdn (register for a free account there to get better download speed). Just try out the hosters we upload to and see which is best for you. If you want to buy a premium account from a file hoster, we recommend Uptobox.

Do I need "cracks" to play the games?

No, every game is DRM-free. Extract the game files, run the setup, install then play!

What if download links for a game go down?

Voting on games is re-enabled after 30 days since they were last uploaded. You can then vote on games to have them re-uploaded for fresh download links.

Any plans to upload Mac or Linux games?

We will only support Windows with no future plans for other operating systems. Also we currently plan to upload English installers only.

How do the new & updated tags work?

When a game (or DLC/goodies) is added/updated; these tags are assigned. They are cleared from game cards after 7 days have passed from being given the tag(s).

Is there a way I can support this site? Do you take donations?

Yes! Click on "Donate" located in the top bar for information on how to donate to us.
